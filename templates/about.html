About the project

This project is a part of the “Russian Artificial Text Detection(RuATD)” research, aimed to study linguistic features that distinguish human-made text from an AI generated one on lexical, morphological and syntactic level.

Using data from the RuATD Dialogue Shared task consisting of human-written texts and texts generated by various language models fine-tuned on different tasks (machine translation, paraphrasing, summarization, simplification and unconditional text generation), we identified a number of features important for evaluating the probability of a text being generated by a machine and trained a classifier to predict the class of text. Click "Try it" button to see how our model works. If you want to learn more about the features and our classifier, check out the Github of the project. Scroll down to have a closer look at some of the most disinct features.

Average number of words in a sentence. Avg feature.

Both T-test and Mann-Whitneyu test have found significant difference in the mean values of Average number of words in a sentence for two classes. Mean value for Human class is 20.7 and 17.9 for Machine class.

Verbs.Morphological feature.

As grammar form, verbs in data are also found to be a significantly important feature. Though classes' means do not differ much, 0.090 for Human and 0.089 for Machine, the difference is still significant, as p-value is less than 0.05 in cases of both statistical tests

Density of punctuation symbols. Frequency feature.

An example of a significant frequency feature. For Human and Machine class the means are 0.031 and 0.28 respectively.
