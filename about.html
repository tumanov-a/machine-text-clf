About the project

This project is a part of the “Russian Artificial Text Detection(RuATD)” research, aimed to study linguistic features that distinguish human-made text from an AI generated one on lexical, morphological and syntactic level.

Using data from the RuATD Dialogue Shared task consisting of human-written texts and texts generated by various language models fine-tuned on different tasks (machine translation, paraphrasing, summarization, simplification and unconditional text generation), we identified a number of features important for evaluating the probability of a text being generated by a machine and trained a classifier to predict the class of text. Click "Try it" button to see how our model works. If you want to learn more about the features and our classifier, check out the Github of the project.
